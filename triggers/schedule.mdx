Un Schedule exécute une tâche sur une base régulière. Vous pouvez définir la planification en utilisant une expression CRON.

## Syntaxe CRON

Les Schedules permettent de déclencher des actions à des moments précis, basés sur une planification CRON.

Ce Schedule s'exécutera à 12h00 tous les dimanches. Vous pouvez obtenir de l'aide avec la [syntaxe CRON](https://crontab.guru/#0_12_*_*_7).

<Note>Les planifications ne sont jamais pas en heure locale, mais en UTC.</Note>

```typescript
export default createSchedule({
  schedule: "0 12 * * 7",
  run: async (ctx) => {
    // Votre code ici
  },
})
```
