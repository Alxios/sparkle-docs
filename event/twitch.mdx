---
title: "Types d'événements Twitch"
description: "Liste des types d'événements Twitch avec leurs descriptions"
---

Cette page répertorie tous les types d'événements Twitch disponibles, avec leurs descriptions associées. Les événements Twitch vous permettent de surveiller et de réagir à différentes actions et interactions sur la plateforme Twitch.

| Name                               | Description                                                                                                                                                           |
| ---------------------------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `channel.update`                   | Un diffuseur met à jour les propriétés de sa chaîne, par exemple la catégorie, le titre, les étiquettes de classification du contenu, la diffusion ou la langue.      |
| channel.follow                     | Une chaîne spécifiée reçoit un abonnement.                                                                                                                            |
| channel.ad_break.begin             | Une pause publicitaire en cours a commencé (BETA).                                                                                                                    |
| channel.chat.clear                 | Un modérateur ou un bot a effacé tous les messages de la salle de discussion (BETA).                                                                                  |
| channel.chat.clear_user_messages   | Un modérateur ou un bot a effacé tous les messages d'un utilisateur spécifique (BETA).                                                                                |
| channel.chat.message_delete        | Un modérateur a supprimé un message spécifique (BETA).                                                                                                                |
| channel.chat.notification          | Une notification indiquant qu'un événement apparaît dans la discussion s'est produit (BETA).                                                                          |
| channel.subscribe                  | Une notification lorsqu'une chaîne spécifiée reçoit un abonné. Cela n'inclut pas les réabonnements.                                                                   |
| channel.subscription.end           | Une notification lorsque l'abonnement à la chaîne spécifiée prend fin.                                                                                                |
| channel.subscription.gift          | Une notification lorsqu'un spectateur offre un abonnement cadeau à un ou plusieurs utilisateurs dans la chaîne spécifiée.                                             |
| channel.subscription.message       | Une notification lorsqu'un utilisateur envoie un message de réabonnement dans une chaîne spécifique.                                                                  |
| channel.cheer                      | Un utilisateur encourage la chaîne spécifiée.                                                                                                                         |
| channel.raid                       | Un diffuseur fait une incursion dans la chaîne d'un autre diffuseur.                                                                                                  |
| channel.ban                        | Un spectateur est banni de la chaîne spécifiée.                                                                                                                       |
| channel.unban                      | Un spectateur est débanni de la chaîne spécifiée.                                                                                                                     |
| channel.moderator.add              | Des privilèges de modération ont été ajoutés à un utilisateur sur une chaîne spécifiée.                                                                               |
| channel.moderator.remove           | Des privilèges de modération ont été retirés à un utilisateur sur une chaîne spécifiée.                                                                               |
| channel.guest_star_session.begin   | L'hôte a lancé une nouvelle session Guest Star (BETA).                                                                                                                |
| channel.guest_star_session.end     | Une session Guest Star en cours s'est terminée (BETA).                                                                                                                |
| channel.guest_star_guest.update    | Un invité ou une place a été mis à jour dans une session Guest Star active (BETA).                                                                                    |
| channel.guest_star_settings.update | Les préférences de l'hôte pour Guest Star ont été mises à jour (BETA).                                                                                                |
| channel.reward.add                 | Une récompense personnalisée en points de chaîne a été créée pour la chaîne spécifiée.                                                                                |
| channel.reward.update              | Une récompense personnalisée en points de chaîne a été mise à jour pour la chaîne spécifiée.                                                                          |
| channel.reward.remove              | Une récompense personnalisée en points de chaîne a été supprimée de la chaîne spécifiée.                                                                              |
| channel.reward_redemption.add      | Un spectateur a échangé une récompense personnalisée en points de chaîne sur la chaîne spécifiée.                                                                     |
| channel.reward_redemption.update   | Une récompense personnalisée en points de chaîne a été mise à jour pour la chaîne spécifiée.                                                                          |
| channel.poll.begin                 | Un sondage a commencé sur la chaîne spécifiée.                                                                                                                        |
| channel.poll.progress              | Les utilisateurs répondent à un sondage sur la chaîne spécifiée.                                                                                                      |
| channel.poll.end                   | Un sondage s'est terminé sur la chaîne spécifiée.                                                                                                                     |
| channel.prediction.begin           | Une prédiction a commencé sur la chaîne spécifiée.                                                                                                                    |
| channel.prediction.progress        | Les utilisateurs participent à une prédiction sur la chaîne spécifiée.                                                                                                |
| channel.prediction.lock            | Une prédiction a été verrouillée sur la chaîne spécifiée.                                                                                                             |
| channel.prediction.end             | Une prédiction s'est terminée sur la chaîne spécifiée.                                                                                                                |
| channel.charity_campaign.donate    | Envoie une notification d'événement lorsque qu'un utilisateur fait un don à la campagne caritative du diffuseur.                                                      |
| channel.charity_campaign.start     | Envoie une notification d'événement lorsque le diffuseur commence une campagne caritative.                                                                            |
| channel.charity_campaign.progress  | Envoie une notification d'événement lorsque des progrès sont réalisés vers l'objectif de la campagne ou lorsque le diffuseur modifie l'objectif de collecte de fonds. |
| channel.charity_campaign.stop      | Envoie une notification d'événement lorsque le diffuseur met fin à une campagne caritative.                                                                           |
| drop.entitlement.grant             | Un droit à une récompense est accordé à un utilisateur.                                                                                                               |
| extension.bits_transaction.create  | Une transaction Bits s'est produite pour une extension Twitch spécifiée.                                                                                              |
| channel.goal.begin                 | Recevez une notification lorsque le diffuseur commence un objectif.                                                                                                   |
| channel.goal.progress              | Recevez une notification lorsque des progrès (positifs ou négatifs) sont réalisés vers un objectif du diffuseur.                                                      |
| channel.goal.end                   | Recevez une notification lorsque le diffuseur met fin à un objectif.                                                                                                  |
| channel.hype_train.begin           | Un Hype Train commence sur la chaîne spécifiée.                                                                                                                       |
| channel.hype_train.progress        | Un Hype Train progresse sur la chaîne spécifiée.                                                                                                                      |
| channel.hype_train.end             | Un Hype Train se termine sur la chaîne spécifiée.                                                                                                                     |
| channel.shield_mode.begin          | Envoie une notification lorsque le diffuseur active le mode de protection Shield Mode.                                                                                |
| channel.shield_mode.end            | Envoie une notification lorsque le diffuseur désactive le mode de protection Shield Mode.                                                                             |
| channel.shoutout.create            | Envoie une notification lorsque le diffuseur spécifié envoie un Shoutout.                                                                                             |
| channel.shoutout.receive           | Envoie une notification lorsque le diffuseur spécifié reçoit un Shoutout.                                                                                             |
| stream.online                      | Le diffuseur spécifié commence un stream.                                                                                                                             |
| stream.offline                     | Le diffuseur spécifié met fin à un stream.                                                                                                                            |
| user.authorization.grant           | L'autorisation d'un utilisateur a été accordée à votre ID client.                                                                                                     |
| user.authorization.revoke          | L'autorisation d'un utilisateur a été révoquée pour votre ID client.                                                                                                  |
| user.update                        | Un utilisateur a mis à jour son compte.                                                                                                                               |

Ces types d'événements vous permettent de créer des fonctionnalités et des réactions personnalisées pour votre bot Twitch dans Sparkle. Vous pouvez surveiller ces événements pour interagir avec votre communauté de manière plus dynamique.
