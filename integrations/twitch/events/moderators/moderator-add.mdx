---
title: moderator.add
description: Cet évènement est émis lorsqu'un modérateur est ajouté sur votre chaine.
---

Exemple d'utilisation :

```typescript
export default createHook({
  event: "moderator.add",
  run: async (_, ctx) => {
    // ...
  },
})
```

### types

Description des propriétés de l'event `moderator.add` :

```typescript
{
  user_id: string // ID de l'utilisateur modérateur ajouté
  user_login: string // Nom d'utilisateur de l'utilisateur modérateur ajouté
  user_name: string // Nom d'affichage de l'utilisateur modérateur ajouté
}
```
