# Context Interface

## Overview

`Context` est une interface clé dans Sparkle, fournissant un accès aux fonctionnalités de Twitch, une base de données, et la capacité d'invoquer des tâches.

## Properties

<ResponseField name="twitch" type="TwitchClient">
  Un objet `TwitchClient` pour interagir avec l'API Twitch. Pour en savoir plus,
  consultez la documentation de [Twitch](/docs/twitch).
</ResponseField>

<ResponseField name="data" type="Database" required>
  Un objet `Database` pour interagir avec la base de données Redis. Pour en
  savoir plus, consultez la documentation de [Database](/docs/database).
</ResponseField>

<ResponseField name="name" type="string[]" required>
  Example: `coin`, `join`, `play`
</ResponseField>

### `twitch: TwitchClient`

- **Description :** Fournit un accès à l'API Twitch pour envoyer des messages et gérer la chaîne et la modération.
- **Type :** `TwitchClient`

### `data: Database`

- **Description :** Permet d'interagir avec une base de données pour stocker et récupérer des informations.
- **Type :** `Database`

## Methods

### `invoke(id: string, payload: any): void`

- **Description :** Permet d'invoquer une tâche avec un identifiant et un payload spécifiques.
- **Parameters :**
  - `id` : Identifiant de la tâche.
  - `payload` : Données à passer à la tâche.

### `invokeUntil(id: string, when: string, payload: any): void`

- **Description :** Permet d'invoquer une tâche répétitivement jusqu'à une condition spécifiée.
- **Parameters :**
  - `id` : Identifiant de la tâche.
  - `when` : Condition déterminant jusqu'à quand la tâche doit être exécutée.
  - `payload` : Données à passer à la tâche.

# TwitchClient Class

## Methods

### `sendMessage({ message }: { message: string }): Promise<void>`

- **Description :** Envoie un message dans le chat Twitch.
- **Parameters :**
  - `message` : Le message à envoyer.

### `get moderation(): TwitchModerationClient`

- **Description :** Accès aux fonctions de modération Twitch.
- **Returns :** `TwitchModerationClient`

### `get channel(): TwitchChannelClient`

- **Description :** Accès aux fonctions de gestion de chaîne Twitch.
- **Returns :** `TwitchChannelClient`

# TwitchModerationClient Class

## Methods

### `ban(data: HelixBanUserRequest): Promise<import("@twurple/api").HelixBanUser[]>`

- **Description :** Bannit un utilisateur de la chaîne Twitch.
- **Parameters :**
  - `data` : Informations sur l'utilisateur à bannir.

### `unban(userId: string): Promise<void>`

- **Description :** Débannit un utilisateur de la chaîne Twitch.
- **Parameters :**
  - `userId` : Identifiant de l'utilisateur à débannir.
